Assignment

Following the described approach, write Javascript programs to solve the problems listed below:

1. Write a program that asks the user for their name and greets them with their name.

2. Modify the previous program such that only the users Alice and Bob are greeted with their names.

3. Write a program that asks the user for an arbitrary number N and prints the sum of the numbers 1 to N

4. Modify the previous program such that only multiples of three or five are considered in the sum, e.g. 3, 5, 6, 9, 10, 12, 15 for N = 17

5. Write a program that asks the user for an arbitrary number N and gives them the possibility to choose between computing the sum and computing the product of 1…, N.

6. Write a program that prints a multiplication table for numbers up to 12.

7. Write a program that prints all prime numbers from 20….. N.

8. Write a program that prints the next 20 leap years.
